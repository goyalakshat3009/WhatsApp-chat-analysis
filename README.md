# WhatsApp Chat Analyzer

A Streamlit-based data analytics project that analyzes exported WhatsApp chats.

## Features
- Total Messages
- Total Words
- Media Shared
- Links Shared
- Most Busy Users
- Word Cloud
- Most Common Words
- Emoji Analysis
- Monthly Timeline
- Daily Timeline

## Tech Stack
- Python
- Pandas
- Streamlit
- Matplotlib
- WordCloud

## Run Locally

pip install -r requirements.txt

streamlit run app.py
